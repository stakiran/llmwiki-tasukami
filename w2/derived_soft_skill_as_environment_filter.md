# 気付き: ソフトスキルは「磨く武器」ではなく「環境選定基準」

ゆるい知的生産の立場から、たすかみのソフトスキル関連語彙を読み直したときに出てきた整理。

## 主張

ソフトスキル（傾聴、説得、共感、対話、リーダーシップなど）は **2 通りに扱える**:

| 立場 | 扱い方 |
| --- | --- |
| ストイック側 | 自分が磨くべき武器。トレーニングで伸ばす |
| ゆるい知的生産側 | **自分の周りに置く人を選ぶための基準**。素養のある人と付き合い、無い人とは距離を置く |

ゆるい知的生産を支持するなら、後者をデフォルトにする。**自分を変えるより環境を選ぶ**。

## 出どころ

- [w1/ファミリア・デバフ](../w1/familiar_debuff.md) — 「できたらラッキー、できなくて当たり前」
- [w1/トーカビリティ](../w1/talkability.md) — 「素養を持つ人の方が少ない」
- [w1/リジェクター](../w1/rejecter.md) — Transfer（自分から距離を置く・逃げる）
- [w1/スラキスト](../w1/slackist.md) — メンタルロード源の「廃棄」

たすかみは「対話できない相手とは盤外戦は成立しない」「素養のない人の方が多い」と冷たく書いている。これを **磨く方向ではなく選ぶ方向** に振り切ったのがこの気付き。

## 含意

- 独立した個人 SE / 上級研究員にとっての **顧客選別軸**: 最初の数回の打ち合わせで「対話の質」を観察する
- パートナーや家族には **ファミリア・デバフを正面突破せず**、別の井戸端メンバーを確保する
- コミュニティ（OSS、Discord、勉強会）の選定が、ソフトスキルの代替インフラになる
- 「営業力を磨かないとダメ」「説得力を鍛えないとダメ」という自己責任論から降りられる

## 派生する問い

- どういうチェックリストで「対話の質」を 1 〜 2 時間で見抜けるか
- ソフトスキルを「磨かない」ことを選んだとき、どこまで仕事が成立するか
- 「自分を磨かない」と「他罰的」は別物。境界はどこか
