# 気付き: 「やってる自覚がない」がゆるい知的生産のシグナル

raw2 の議論と w1 の Inboxer の定義を重ねたら出てきた検出器。

## 主張

ゆるい知的生産が成立しているかどうかは、**本人にやってる自覚があるかどうか** で見分けられる。

| 状態 | 本人の自覚 |
| --- | --- |
| ゆるい知的生産が回っている | **「タスク管理」「知的生産」をやっている自覚がない**。Slack で雑談している、Scrapbox に書き散らしている、で終わる |
| ストイック側に倒れている | 「私は知的生産をやっています」「私はタスク管理をやっています」と自覚的にアピールする。手法名や略語を語る |

つまり **自覚を売り物にしているかどうか** で、本物と「教える人」を仕分けできる。

## 出どころ

- [raw2/ゆるい知的生産.md](../raw2/ゆるい知的生産.md):
  - kuuote「Scrapbox を使って情報溜めてる時点で『知的生産』なのかもしれない」
  - nishio「実際にやってる人がやってること」と「教える人が教えてること」が食い違う
- [w1/inboxer.md](../w1/inboxer.md):
  - 「Inboxer は極めて合理的であり、タスク管理を行っているとの自覚はありません」
- [w1/active_lazy.md](../w1/active_lazy.md):
  - 「アクティブレイジーは怠け者なので、他の戦略なんてできやしません」

両者は「**できている人ほど自覚していない**」という同じ構造を別の言葉で書いている。

## 含意

### 自分への適用
自分が「私は LTM をやっている」「私は ETM を運用している」と自覚的に語り始めたら、警戒する。**手法を語ることが目的化** している兆候かもしれない。

逆に「気付いたらノートが何百枚溜まっていた」「気付いたら週に何時間も雑談していた」なら、ゆるい知的生産が機能している可能性が高い。

### 他者を見るとき
誰かが「私は X 法をやっています」と熱心に語る場合、その人は:

1. 本物だが、教える側に回ろうとしている（マナー講師化の入り口）
2. 自覚的に頑張っている（持続性に疑問）
3. やってる自分を自慢したい（raw2 で nishio が指摘した型）

のいずれかの可能性が高い。**実際にやれている人ほど、聞かないと話さない**。

### 学ぶ相手の選び方
新しい手法や考え方を学ぶときは:

- 「これをやれば誰でもできます」と語る人より、「気付いたらこうしてました」と振り返って言う人を優先する
- 体系化された講座・本より、雑なログや雑談の方が価値が高い場合が多い

これは raw2 の「井戸端の方がアクティブ」と同じ構造。

## 注意点（この検出器の限界）

- **本物だけど自覚的に体系化している人** もいる（nishio や masui や rashita 自身もそう）。彼らは「教える人」と本物の境界に立っており、検出器は誤判定する
- 自覚を語らないことが「謙遜パフォーマンス」になっている場合もある
- 短期で見ると判定できない。長期の生産物の質と量で見るしかない

つまりこの検出器は **強い陽性ではなく、弱い陰性として** 使うのが妥当: 「自覚を熱心に語っていない」=「ゆるい側の可能性が残る」程度。

## 派生する問い

- 自分が体系化に向かい始めたとき、どこで止めるか
- たすかみ自体が「教える人」サイドに立っている本だが、それでもゆるい知的生産を扱えているのはなぜか（おそらく **盤外戦・スキッパー・スラキスト・Inboxer・No ABCD** など「やらないこと」を厚く書いているから）
- LLM が自覚なしに支援する時代、この検出器はどう変質するか
