# 3A — プロジェクトタスク管理の本質

[プロジェクトタスク管理](project_task_management.md) の本質を 3 つに集約した枠組み。

| 英語 | キーワード | 解説 |
| --- | --- | --- |
| **Assign** | アサイン | 人にタスクを割り当てること |
| **Adjust** | 調整 | タスクの属性値を変えること |
| **Alternate** | 全体と詳細 | タスク全体の俯瞰と特定タスクの注視を行き来すること |

各要素の詳細は専用ページへ:

- [Assign / アサイン](assign.md)
- [Adjust / 調整](adjust.md)
- [Alternate / 俯瞰と注視](alternate.md)

## まとめると

> 結局やりたいことは進捗を知りたい。ただそれだけ

3A はそのための営みと言ってもよい。3A を上手くやれればタスク管理は要らないし、やれるなら別に管理者も要らない。
