# 「割り込みと脱線」

chap4 の 4 つ目。タスクをスムーズに遂行できない要因の一つ「ワープ」を扱う章。

## 要約

- **[ワープ](warp.md)** = 割り込み（外部から）と脱線（自分から）の総称
- 弊害は **パフォーマンス阻害** と **コンテキストスイッチング**
- 処理は [SWAP マトリックス](warp.md) で 4 パターン: Active Strong / Active Weak / Passive Strong / Passive Weak
- 各パターンには **対処** と **防止** がある
- Passive Strong（誘惑に負けない）の対処は難しい。**[Explicit Attention](explicit_attention.md)** と **[リズムリマインド](rhythm_remind.md)** の 2 手段

## 章で導入される主な概念
- [ワープ](warp.md) — 割り込みと脱線、SWAP マトリックス、4 パターン
- [Explicit Attention](explicit_attention.md) — 書きながら行動して注意を明示する
- [リズムリマインド](rhythm_remind.md) — 染み付いた生活リズムからのツッコミに頼る

## 引用しておきたい箇所
- 「**ひとまずの判断や返事でさえも無視できるなら脱線、無視できないなら割り込み**」
- 「**プラスを得るよりもマイナスを避けます**」（Passive Strong の防止）
- 「**書くことで注意の対象を明示的につくる**」（Explicit Attention）
- 「**自分を軌道修正してくれるような言い回しや雰囲気** だったり」（バランスの難しさ）
- 「リズムリマインドは **誰にも見せず知らせずに働きかけてもらえる** 数少ない手段」
