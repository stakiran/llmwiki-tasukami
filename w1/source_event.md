# 「イベント」

[オルタスク](altask.md) のひとつ「予定」を本格定義する章。

## 要約

- **[イベント](event.md)** = いわゆる「予定」。開始時間と開始場所の少なくとも片方を拘束され、終了タイミングも指定されているか調整が必要なもの
- イベントの本質は **儀式**。面倒くさい事柄
- 拘束パターンで **6 種類** に分けられる: 予定全般 / 割り込み的開催 / リモートワーク / 閉館時間を持つ場所 / エンドレス / 不明瞭な終わり
- 各種類で対処が違う: カレンダー / 信頼関係と読み合い / 自律性と盤外戦 / 自分の調整 / 予防 / メリハリ制・明示的提案・ダミーイベント

## 章で導入される主な概念
- [イベント](event.md) — オルタスク「予定」の本格定義

- 「自分が忘れていたとしても誰かが or 何かが思い出させてくれれば良い」
- 「**ミュートデイ** ――一日全く打ち合わせがない日が当たり前にあるかどうか」
- 「タイプ 5（エンドレス）のイベントは地獄でしかなく、タスク管理でどうにかなるものではない」
