# RSAF サイクル

タスク管理ツールを使って [忘迷怠](boumeitai.md) を担保するための行動ループ。

| 段階 | 内容 |
| --- | --- |
| **R**ead | ツールを見る |
| **S**elect | タスクを選ぶ |
| **A**ct | 行動する |
| **F**eedback | 行動後ツールに戻り状況を更新する |

一巡したらまた Read から始まる。

## 回すための前提

このサイクルを回すには、ツールが [動線](dousen.md) に組み込まれている必要がある。Read が成立しなければ何も始まらない。

## カレンダーは Feedback 不要

カレンダーは過去の予定を更新しなくてよい（放置でいい）ため、最後の Feedback が省ける。この **ちょっとした手間の無さ** がカレンダーが親しまれる理由の一つになっている。
