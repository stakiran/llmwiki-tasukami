# 「パートナータスク管理」

[3P 分類](3p.md) のうち Partner を端的に整理する章。前章（プロジェクト）と同様、本書の主題は個人タスク管理だが、パートナータスク管理についても本質を整理しておく。

## 要約

- パートナーと共同で行う、日常生活全般のタスク管理。家事、育児、引っ越しや旅行のような家庭イベント
- 成立しづらい根本理由は **[ファミリア・デバフ](familiar_debuff.md)** — 親しい間柄ゆえに面倒くさい諸活動はサボりがちになる
- ツールの変遷: 口頭/電話 → メール → LINE → Slack/Discord（**夫婦 Slack**、**カップル Slack**、**ひとり Slack**）→ TimeTree、Cross などの専用アプリ
- 本質は **3C**: [Chat per Task / Calendar View / Concept](3c.md)
- 実はタスク管理よりも [盤外戦](bangaisen.md) の方が重要。意識的に盤外戦のための時間を捻出する必要がある
- 盤外戦の素養として [トーカビリティ](talkability.md)（対話できる素養）が必須

## 章で導入される主な概念
- [ファミリア・デバフ](familiar_debuff.md)
- [3C](3c.md) — Chat per Task / Calendar View / Concept
- [盤外戦](bangaisen.md) — 本章で本格定義 + 6 つのテクニック
- [トーカビリティ](talkability.md)

## 引用しておきたい箇所
- 「**タスクの可視化は、思いやりのためにも重要です**」
- 「**タスク管理全般が盤外戦にかかっています**」
- 「**そもそもパートナータスク管理を行えるだけの素養を持つ人の方が少ない**」
