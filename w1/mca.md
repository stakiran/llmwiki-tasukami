# MCA — 管理の 3 ニュアンス

「●●管理」と呼ばれるものには 3 種類のニュアンスがある。

| 名前 | 解説 | 一言で |
| --- | --- | --- |
| **Management** | あらゆる手段を使って「うまくやる」 | 総合格闘技 |
| **Control** | 計画や基準のとおりに動いているかを「保証する」 | ギャップを埋める |
| **Administration** | 方向性や仕組みを定めるなど「舵取りをする」 | トップダウンな支配 |

元ネタの参考リンクは原稿に記載あり（note.com / brevis.exblog.jp）。

## 「●●管理」をこの軸で並べてみる

| 名前 | MCA | 解説 |
| --- | --- | --- |
| 進捗管理 | Control | 期限までに完了できるか、進行ペースは問題ないか |
| 完了管理 | Control | 何ができたのか、何を終えたのか |
| 見積管理 | Control | 完了にかかる時間、正確な予測の方法 |
| 会議管理 | Management | どんな会議体を設定するか、緊急で誰と何を打ち合わせるか |
| 稼働管理 | Administration | 人材リソースの定義、消費の判定（典型例: 工数管理） |
| 在庫管理 | Administration | バックログ、いつかやるリスト、定期的な棚卸し |

世間の「管理」は **Control と Administration が多い**。

## なぜタスク管理は Management なのか

タスク管理の英名は Task Management。Control でも Administration でもない。

- 「人それぞれ」「状況次第で最適解が変わる」という性格と整合
- 要するに **うまくやれれば何でもいい**
- Control や Administration は手段としては有効だが、それだけではない
