# 3C — パートナータスク管理の本質

[パートナータスク管理](partner_task_management.md) の本質を 3 つに集約した枠組み。

| 名前 | 解説 |
| --- | --- |
| **Chat per Task** | タスクごとのチャット（やり取り） |
| **Calendar View** | カレンダーによる俯瞰 |
| **Concept** | 細かいコンセプトで差別化 |

## Chat per Task — タスクごとのやり取り

パートナーは:

- 気軽にやりとりできる
- でもやりとりだけだとタスク管理できない
- 遠慮も要らないし仕事でもないので、いちいち面倒くさいタスク管理なんて基本的にしたくない

この微妙な機微を満たせるのが Chat per Task。**タスクごとにチャットを設ける** バランス。

タスクの粒度はパートナー間で決める:

- 細かい: 「時計を引き取りに行く」タスクを新しくつくる
- 粗い: 「2024/05/20 の週」タスクをつくり、その配下でチャット撃つ
- 最も粗い（LINE）: 場が一つしかなく日常の複数タスクが混ざる、過去情報も探しづらい

対話しながら自分達に合う粒度を探る。

## Calendar View — カレンダーによる俯瞰

タスク管理のタの字を知らない人でも、カレンダーは知っている。馴染みがあって使いやすい。

二人共用のカレンダーに共同で予定を書き込む。スマホからいつでも見られる。

> タスクの可視化は、思いやりのためにも重要

お互い何をしているか（するつもりなのか）がわかれば、先回りして配慮できる。「外せない時間があるなら予定として入れておけ、入れてなかったら割り込めると解釈するからな」のようなルール化もアリ（親しき仲にも礼儀あり）。

## Concept — 細かいコンセプトで差別化

正解は一つではない。理想は自分達で模索だが、仕事ではないので努力しづらい → ツール側がコンセプトを打ち出して利用者と出会いを生む。

| ツール | コンセプト |
| --- | --- |
| **TimeTree** | 予定の共有と相談（カレンダー寄り） |
| **Cross** | 夫婦のための ToDo 共有（タスク寄り、ライフイベント対応の「やることセット」あり） |
| **家事の名は** / **Yieto** | 家事の見える化に特化 |

現時点で「パートナータスク管理」というジャンル名はまだ存在しないが、共有アプリ・家事分担アプリの呼称はある。マニア以外でも使えるデザインを実現したアプリが出れば、一気に広まる可能性。
