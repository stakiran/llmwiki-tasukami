# 「個人タスク管理を眺める」

[3P 分類](3p.md) のうち Personal を眺める章。[個人タスク管理](personal_task_management.md) の主要なパターンを観点別に紹介する。

## 要約

- そもそもタスク管理するかしないかは特性と状況による。タスク管理は **運動** のようなもので、面倒くささに耐えられるかが鍵 → [トレメント](trement.md)
- カバーする観点は 3 つに集約: [忘迷怠](boumeitai.md)（確実性）、[調動脈](choudoumyaku.md)（納得性）、[熱夢集](netsumushuu.md)（パフォーマンス）
- 忘迷怠の鍵は [動線](dousen.md)。ツールを動線に組み込むことで [RSAF サイクル](rsaf.md)（Read → Select → Act → Feedback）が回る
- どこまで管理するかで [浅い/深いタスク管理](shallow_deep.md) に二分される
- ツールは [アナログ / デジタル](analog_digital.md) で性質が大きく異なる
- 多様な [戦略](strategies.md)（時間/話題/一箇所/繰り返し/脳内）と [スタンス](stances.md)（量を減らす/省力化/盤外戦）がある

## 引用しておきたい箇所
- 「**タスク管理は、たとえるなら運動のようなものです**」
- 「**ツールを動線に組み込む必要があります**」
- 「**集中は制約のかけ方をいかに工夫するかにかかっています**」
- 「**好きこそものの上手なれ**」（調動脈の継続論）
- 「**深いタスク管理の目的が打開と前進だから**」
