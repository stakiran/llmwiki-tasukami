# 個人タスク管理

個人が自分のタスクを自分で管理する形のタスク管理。[3P 分類](3p.md) の Personal に対応し、**本書の主題**。

## 本質: 人それぞれ

人によって何を重視するかが大きく違う。例:

- A さん: 本当に重要なタスクだけをこなしたい
- B さん: 掃除や爪切りやゴミ捨てなど日常の家事まで完璧にコントロールしたい
- C さん: 予定表に詰め込んだ予定をひたすら消化していくマシーン

A・B・C の各タスク管理は、互いには通じない。

同じ人でも変わる:

- D さん: 平日は C さんのような予定消化マシーン、休日は A さんのように重要タスクを優先順位付けてこなす
- E さん: 平日は予定にとらわれず自分の裁量でのびのび、休日は C さん的に予定でアクティブに過ごす

## それでも傾向はある

「予定に頼る」スタイルは多くの人が知っているし使っている。管理職など忙しい人は、明示的に抗わなければこのスタイルになりやすい。本書ではこの傾向も整理して取り上げる。

## タスク管理の周辺こそ重要

人間は怠ける・迷う・忘れる生き物。集中力やモチベーションも有限。タスク管理が完璧でも、タスクに取り組む人間自身が動けなければ意味がない。

> タスク管理は、実はタスク管理の周辺の要素こそが重要
