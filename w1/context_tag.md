# コンテキストタグ (Context Tag)

タスクにつけておく「いつ実行できるのか」「どんな性質なのか」を示す用のタグ。著者の造語。[コンテナ](container.md) の応用。

## 例

- **「通勤中」タグ** — 電車内・バス内でこなせるタスクにつける。スマホで通勤中タグの一覧を見ることで車中にタスクをこなせる
- **「あとで読む」タグ** — 単純で頻出
- **「夫がいないとき」タグ / 「上司がいるとき」タグ** — 人間関係の制約

特に忙しい人や対人関係が多い人は、この手の制約が色々あるため、コンテキストタグでスキマ時間にタスクを消化しやすくなる。

## 「文脈（Context）」とは別概念

紛らわしいが、[調動脈](choudoumyaku.md) で扱う **文脈（Context）**（タスクの前提・背景・真意）とは別概念。コンテキストタグは「コンテキスト = いつ・どこで・どんな状況で実行できるか」を表す札。
