# 「タスク管理の戦略」

[戦略](strategies.md) — どんなやり方や考え方で望むかという方針 — を 5 カテゴリ・9 種類に分けて詳述する章。本書の主題である [個人タスク管理](personal_task_management.md) に戻る章でもある。

## 要約

- **時間で区切る** — メリハリ系。[Dailer](dailer.md) / [Calendarer](calendarer.md) / [Slotter](slotter.md)
- **話題で区切る** — 寝かせる系。[Issueist](issueist.md) / [Topician](topician.md)
- **一箇所にまとめる** — 手間暇かけない系。[Richild](richild.md) / [Monolith](monolith.md)
- **繰り返しの力でまわす** — リズム系。[Robot](robot.md) / [Tracker](tracker.md)
- **自分の脳内が正義** — ツール最小系。[Inboxer](inboxer.md)

## 章で導入される主な概念
- 9 戦略: [Dailer](dailer.md) / [Calendarer](calendarer.md) / [Slotter](slotter.md) / [Issueist](issueist.md) / [Topician](topician.md) / [Richild](richild.md) / [Monolith](monolith.md) / [Robot](robot.md) / [Tracker](tracker.md) / [Inboxer](inboxer.md)
- 前提知識: [物タスク](object_task.md) / [ルーチンタスク](routine_task.md)
- 重要な造語: [オプラン](oplan.md) / [アクティブレイジー](active_lazy.md)

## 引用しておきたい箇所
- 「**Dailer はタスクへの取り組み方に裁量があることが前提**」
- 「**リマインダーは「ひとりだけで」すっぽかしを防ぐための最後の砦**」
- 「**睡眠以外の予定がギッチギチの日が一週間続いても生活が持続する状態** が（Calendarer の）適正」
- 「**生活において必要なルーチンタスクのすべてが集まっており、しかも適切な頻度で出現するという安心感**」（緩いロボット）
- 「**こういう微調整ムーブを息するように行えるかどうかが、Robot 定着の分水嶺になる**」
- 「**タスク管理では形骸化が常態化すると、すぐに破綻します**」（Tracker）
