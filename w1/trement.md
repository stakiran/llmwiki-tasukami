# トレメント (Trement)

**Training Element** の略。タスク管理に伴う本質的な面倒くささを指す造語。

タスク管理は運動に似ている。筋トレでもスポーツでも軽いアクティビティでも、体を動かすという面倒くささから逃げられない。タスク管理にも同じ性質があり、それがトレメント。

> タスク管理は、たとえるなら運動のようなものです

## トレメントとの相性が分かれ目

タスク管理を続けられるかは、トレメントを好きになれるか・嫌いでも続けられるかにかかっている。

### タスク管理が要らない人
- 頭の中だけで処理できる人 → そもそもツールが要らない（[ツールの公理](axioms.md) 不成立）
- 仙人のように生活に余裕がある人
- 言われたことをやるだけの人 → 解釈が要らない
- バイタリティがあり「まあ何とかなる」と楽観できる人 → 部屋やデスクトップが散らかっていても回せる
- 慌ただしい状況に身を投じている人 → 臨機応変な取捨選択と会話が最重要で、タスク管理の出番が無い

> 悩むのは暇だからだ（カーネギー『道は開ける』より）

### タスク管理が要る人
- 頭の処理能力が低い → 情報を外に出してメンテする必要に駆られる
- 神経質 → ルーチンをストイックに構築する
- 几帳面、自分のリズムやペースに従いたい → 生活の質も上げられる

## するか、しないかの目安

> 現状すでに私生活で（仕事で強要されるのではなく自由意思で）タスク管理またはそれに近しいことをしているかどうか

- している人 → 必要。本書のヒントを漁って合うものを探す
- していない人 → 不要。本心で要らないと思っているなら無理して始めなくてよい

## トレメントが苦手でもできる

トレメントが苦手でも工夫されたやり方はある。

- **ポモドーロ・テクニック** — 25 分集中 + 5 分休憩のリズム。制約のかけ方の工夫
- **ゲーミフィケーション系ツール** — Habitica、Chill Pulse など

> 自炊と同じ。料理人のようにつくれずとも、野菜と肉を切って鍋に放り込むだけでもそれなりのものはつくれる。

部分的に取り入れるだけでも構わない。
