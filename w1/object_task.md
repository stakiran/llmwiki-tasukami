# 物タスク

タスクを想起するもの、またはタスクの構成要素となっているもの。著者の造語。

## 例
- 「壊れた時計」 → 「時計を修理する」タスクを生み出す
- 「上司の顔画像」 → 「上司から頼まれていたこと」を思い出す

物タスクかどうかは、その人の解釈による。同じ人でも状況が違えば変わる。

## タスクとの違い

[タスク](task.md) は「やることが文章化されたもの」（[ツールの公理](axioms.md)）。物タスクは「タスクになるかもしれない物」。

## どこで使われるか

[Richild](richild.md) 戦略の対象が物タスク。

- タスクをまとめる: 付箋でディスプレイに貼ること
- 物タスクをまとめる: 「外で済ませたい用事の物（壊れた時計、ゴミ袋、おめかしセット）」を玄関に置くこと

普段から「外で済ませたいものは玄関に置いておけ」というルール化のニュアンス（その場限りで行うのではなく）。
