# 「タスク管理を噛み砕く」
- Zenn Book として書かれている
- エントリーポイントは config.yaml である
    - chapters が目次を示しており、章単位で対応ファイルを並べている
    - 上から順番に 1 章、2章、……と対応している
    - 例: about が 1 章、chap1 が 2 章、view_taskmanagement が 3 章
    - また chapXX は区切りページであり、中身はない
- 各ファイルは Markdown で書かれている
    - 画像のリンク先は、実際は `assets/xxxx.jpg` や `assets/xxxx.gif` のように読み替えること
